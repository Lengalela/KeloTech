HTML Tutorial

HTML (HyperText Markup Language) is the standard markup language used to create web pages.

HTML Basic

HTML documents consist of a series of elements, which are represented by tags.

            <html>
            <head>
                <title>Page Title</title>
            </head>
            <body>
                <h1>My First Heading</h1>
                <p>My first paragraph.</p>
            </body>
            </html>

HTML Elements

HTML elements are represented by tags, which are surrounded by angle brackets.

  • <p> - defines a paragraph of text
  • <h1> - defines a large heading
  • <img> - defines an image

CSS Tutorial

CSS (Cascading Style Sheets) is a styling language used to control the layout and appearance of web pages.

CSS Basic

CSS is used to separate the presentation of a document from its structure.

CSS Selectors

CSS selectors are used to target specific elements on a web page.

  • body - selects the body element
  • h1 - selects all h1 elements
  • .header - selects all elements with the class "header"

JavaScript Tutorial

JavaScript is a programming language used to add interactivity to web pages.

JavaScript Basic

JavaScript is used to create dynamic effects on web pages.

console.log("Hello World!");

JavaScript Variables

JavaScript variables are used to store values.

  • var - declares a variable
  • let - declares a block-scoped variable
  • const - declares a constant variable
