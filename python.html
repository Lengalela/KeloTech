Previous Lesson Recap

JavaScript is a powerful programming language used to make websites interactive. It enables dynamic responses to user actions, making web pages more engaging and functional. Some real-world applications include web-based games, interactive maps, online form validation, and dynamic website elements. In this lesson, you practiced writing and modifying basic JavaScript programs, including alerts and user prompts.

1. What is a JavaScript Function

  • Definition: A function is a block of reusable code that performs a specific task. Functions help programmers write cleaner and more efficient code.
  • A function is like a recipe: You write it once and use it multiple times instead of repeating steps.
  • Examples: A calculator function to add numbers, a greeting function that welcomes users.

2. Real-World examples

  • Form Validation: Checking if a user has entered a valid email before submitting a form.
  • Automated Messages: Displaying a welcome message when someone visits a website.
  • Math Calculations: A function that adds or multiplies numbers in an online calculator.
  • Event Handling: Running specific code when a button is clicked.

3. Examples

1. What are Arrays and Objects

Arrays:

  • Definition: An array is a collection of items (values or variables) stored in a single variable. Arrays are used to store multiple values in a single variable, making it easier to work with large datasets.
  • Structure: Arrays are ordered collections, indexed by integers (starting at 0).
  • Example:
let fruits = ["Apple", "Banana", "Cherry"];
                console.log(fruits[1]); // Outputs: Banana

Objects:

  • Definition: An object is a collection of key-value pairs. Each key (also called a property) is unique, and it is associated with a value.
  • Structure: Objects are unordered collections of key-value pairs, where the key is a string, and the value can be any data type (e.g., string, number, array, function).
  • Example:
let person = {
                    name: "John",
                    age: 30,
                    job: "Developer"
                };
                console.log(person.name); // Outputs: John

2. Examples

Example 1: Working with Arrays

<!DOCTYPE html>
                <html>
                <head>
                    <title>Array Example</title>
                </head>
                <body>
                    <h2>Array Example - List of Fruits</h2>
                    <p id="demo"></p>
                    <script>
                    let fruits = ["apple", "banana", "cherry"];
                    fruits.push("orange"); // Add a new fruit
                    document.getElementById("demo").innerHTML = fruits;
                    </script>
                </body>
                </html>

Example 2: Working with Objects

<!DOCTYPE html>
                <html>
                <head>
                    <title>Object Example</title>
                </head>
                <body>
                    <h2>Object Example - Car</h2>
                    <p id="demo"></p>
                    <script>
                    let car = {
                        make: "Tesla",
                        model: "Model S",
                        year: 2022
                    };
                    car.color = "red"; // Add a new property
                    document.getElementById("demo").innerHTML = car.make + " " + car.model + " (" + car.year + "), Color: " + car.color;
                    </script>
                </body>
                </html>

Example 3: Array with Objects

<!DOCTYPE html>
                <html>
                <head>
                    <title>Array of Objects Example</title>
                </head>
                <body>
                    <h2>Array of Objects - Students</h2>
                    <p id="demo"></p>
                    <script>
                    let students = [
                        { name: "John", age: 16 },
                        { name: "Jane", age: 15 },
                        { name: "Bob", age: 17 }
                    ];
                    
                    students.push({ name: "Alice", age: 18 }); // Add new student
                    
                    let text = "";
                    for (let i = 0; i < students.length; i++) {
                        text += students[i].name + " is " + students[i].age + " years old.<br>";
                    }
                    
                    document.getElementById("demo").innerHTML = text;
                    </script>
                </body>
                </html>

Exercises

1. Exercise 1: Create an Array of Colors

Task: Create an array of at least 5 colors. Add a new color to the array using the push() method, then display all the colors using a for loop.

<!DOCTYPE html>
                <html>
                <head>
                    <title>Array of Colors</title>
                </head>
                <body>
                    <h2>Array of Colors</h2>
                    <p id="demo"></p>
                    <script>
                    // Create an array of colors
                    let colors = ["red", "blue", "green", "yellow", "orange"];
                    // Add a new color to the array
                    colors.push("purple");
                    // Display all the colors
                    let colorList = "";
                    for (let i = 0; i < colors.length; i++) {
                        colorList += colors[i] + "<br>";
                    }
                    document.getElementById("demo").innerHTML = colorList;
                    </script>
                </body>
                </html>

2. Exercise 2: Create an Object for a Book

Task: Create an object that represents a book with properties such as title, author, and year. Add a new property called genre and display the book information.

<!DOCTYPE html>
                <html>
                <head>
                    <title>Book Object</title>
                </head>
                <body>
                    <h2>Book Information</h2>
                    <p id="demo"></p>
                    <script>
                    // Create an object representing a book
                    let book = {
                        title: "JavaScript for Beginners",
                        author: "John Doe",
                        year: 2020
                    };
                    // Add a new property to the object
                    book.genre = "Programming";
                    // Display the book information
                    let bookInfo = "Title: " + book.title + "<br>" +
                        "Author: " + book.author + "<br>" +
                        "Year: " + book.year + "<br>" +
                        "Genre: " + book.genre;
                    document.getElementById("demo").innerHTML = bookInfo;
                    </script>
                </body>
                </html>

Summary

  • Arrays: A collection of values stored in a single variable. Items are accessed using an index.
    • Example: let fruits = ["Apple", "Banana", "Cherry"];
  • Objects: A collection of key-value pairs, where each key has a corresponding value.
    • Example: let person = {name: "John", age: 30};

Key Concepts:

  • Array Operations: Add values using push() and display using loops.
  • Object Operations: Add properties and display using dot notation.
